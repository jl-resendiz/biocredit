# Parameter Registry

Comprehensive documentation of every numeric parameter in both models plus the early warning system. Serves as audit trail for reviewers and cross-model consistency check.

**Models covered:**
- **ABM** = Formation-level Agent-Based Model (`scripts/run_abm.py` = PRIMARY; `src/model/formation_model.py` = prototype)
- **GLV** = Generalised Lotka-Volterra mean-field model (`src/model/glv_formation.py`)
- **EWS** = Early Warning Signal system (`src/earlywarning/signals.py`)

**Parameter derivation script:** `scripts/derive_parameters.py`

### Flagged discrepancies

1. **Power-law alpha (4.0 vs 1.55)**: These are intentionally different parameters. Alpha=4.0 in ABM controls *input* demand concentration (how strongly demand is directed to top OTGs). Alpha=1.55 in GLV is the *output* Zipf exponent (how transactions actually distribute across OTGs after stochastic matching). The ABM's stochastic process moderates the 4.0 input concentration to approximately 1.55 in the output, which is why the GLV uses the empirical outcome directly.
